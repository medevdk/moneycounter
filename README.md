## Money Counter ##

### Electron app ###

So far only works on my MacOS, not yet on my Fedora Linux.
Very old project.

#### Install ####
```
cd ~
git clone git@github.com:medevdk/moneycounter
cd moneycounter
```

#### Install electron ####
```
npm install --save-dev electron
npm start
```

Or if you do not want to install electron
```
npx electron .
```
